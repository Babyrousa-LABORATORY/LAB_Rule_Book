# バビルサ研究所ゲームパート概説(判定編) {#title}

2025/3/31

本ページでは、ゲームパートでの判定について説明する。判定は基本的にD20を利用して行われ、基礎パラメータの「修正値」、特殊パラメータの値と組み合わせて判定される。パラメータについて知りたい者は、[「ステータス・パラメータ編」](../data_catalog/parameters.md)を参照のこと。

## 判定のきほん {#judge_basic}

基本には、判定は「目標値」と「達成値」の比較で行われる。目標値は、これから行う行為の難易度を数値化したもの。また、達成値は、クリーチャーが実際にその行動を試みて、挙げることができた成果を数値化したものである。いずれも、高ければ高いほど高難易度/上出来であると言える。
達成値は、以下の計算式より算出される。

`(受動回避力) = 1d20 + (基礎ステータス修正値) + (特殊ステータス)`

なお、「修正値」は、基礎ステータスの実際の値より10を引き、2で割ったものを指す(小数点以下切り下げ)。

`(基礎ステータス修正値) = ((基礎ステータス) 10) / 2`

また、以降の判定は、以下のように記載することにする。

`(基礎ステータス名)「(特殊ステータス名)」判定`

(例)
    知力「コンピュータ」判定
    →`1d20 + (知力修正値) + (コンピュータ値)`
    判断力「精神抵抗」判定
    →`1d20 + (判断力修正値) + (精神抵抗値)`

### つかいどころ {#judge_use_case}

#### 命中判定 {#hit_judge}

敵味方を問わず、攻撃時には必ず「命中判定」が必要となる。
これは、振りかぶった攻撃がちゃんと敵に当たったかを判定するものになる。
これらは、攻撃のタイプによっても異なるが、例えば近接格闘時には、**筋力「格闘」判定**が必要になる。

`1d20 + (筋力修正値) + (格闘値)`

##### 命中判定になりうる判定 {#hit_judge_example}

筋力「格闘」
:   `1d20 + (筋力修正値) + (格闘値)`
    格闘武器で相手を叩く際に利用する。格闘はおおむね全ての動物で利用される攻撃手段。それもあってか、大半のアニマル・ウェポンでも利用される、最も汎用的な判定といえる。

筋力「投擲」
:   `1d20 + (筋力修正値) + (投擲値)`
    投擲武器または一部のアイテムを投げつける際に利用する。投げと言う攻撃はあまり一般的な技ではないため、苦手とするアニマル・ガールは少なくない。

敏捷力「射撃」
:   `1d20 + (敏捷力修正値) + (射撃値)`
    弓や銃火器などで相手を射抜く際に判定する。投げよりも一般的ではない、あるいは動物でいるうちはまずあり得ない動作であるためか、ウェポン・マスターでもない限りは鍛える機会すらないとされている。故に、しっかりこなせるフレンズは畏怖の対象になると言われているとか。

魔力「魔法」
:   `1d20 + (魔力修正値) + (魔法値)`
    魔法(！)で相手を攻撃する際に利用する。
    バフやデバフ、状態異常付与時にも利用される機会が多い。
    サンドスター由来の物質故、厳密には「魔法」と呼ぶのは適切ではないとも言われるが、基本的に超常現象が発生する場合は便宜上この言葉が利用されることが多い。
    *一部のアニマル・ガールはこの判定でアニマル・ウェポンをふるう*ので、攻撃型にしたい場合はよく確認すること！

#### 状態異常抵抗判定 {#struggle_judge}

状態異常にかかりそうになった時は、抵抗判定を求められる。
これに成功すれば、君は迫り来る敵の攻撃を回避できるわけだ。
やはりこれらも喰らう攻撃のタイプによって異なるが、例えば「混乱」という状態異常にかかるときは、判断力「精神抵抗」判定が必要になる。

`1d20 + (判断力修正値) + (精神抵抗値)`

こうなる理由は、「混乱」することが対象のメンタルに関わるためである。
同様に、火傷などのフィジカルに関わる判定では、耐久力「肉体抵抗」判定が必要になる。

##### 抵抗判定になりうる判定 {#struggle_judge_example}

判断力「精神抵抗」
:   `1d20 + (判断力修正値) + (精神抵抗値)`
    メンタル系状態異常に当たる技を喰らうと、この判定が求められる。
    人生、精神を揺さぶられることなんてしょっちゅうではあるが、そういうときこそ冷静に状況を分析する力が必要になってくるものだ。
    メンタル系に当たる状態異常には以下のようなものがある。
    - 混乱
        - 効果時間中、行動がランダムで決定。基本的には行動不能だが、最悪自分にダメージを与えてしまうので注意。
    - 魅了
        - 効果時間中、行動不能。たまに自分が先に飛ばした攻撃を自分が喰らってしまうことも。味方に攻撃してもらうと治ることがある。
    - 封印
        - 効果時間中、スキルが一切使用不可能。

耐久力「肉体抵抗」
:   `1d20 + (耐久力修正値) + (肉体抵抗値)`
    フィジカル系状態異常に当たる技を喰らうと、この判定が求められる。
    ダメージを喰らうと同時に発生するケースが多く、治療の機会もかなり多く求められる。
    フィジカル系に当たる状態異常には以下のようなものがある。
    - 毒、出血、火傷
        - 効果時間中、スリップダメージを受け続ける。
    - 氷結
        - 効果時間中、行動不可能。加えて、次に喰らう攻撃のダメージが2倍となる。
    - 鼓膜破壊
        - 効果時間中、「音が聞けなくなる」。……効果が薄そう?そうとも言い切れないだろう。

運「その他抵抗」
:   `1d20 + (運修正値) + (その他抵抗値)`
    シックネス系の状態異常にかかるとこの判定が求められる。
    基本的にこの判定が求められる事は滅多にない。  
    **この判定が求められる時はマジで研究員の命に関わる時なので、絶対に失敗しないように！！！**

#### 探索判定 {#search_judge}

探索シーン時に利用する。
アイテムを探したり、少ない情報からヒントを探ったりといった行動をする際には、相応の「知力」が求められる。
なので、探索シーンで利用される判定は知力系の判定であることが多い。
もちろん、シナリオの展開次第では、自己申告で知力以外の判定を利用しても構わない。

##### 探索判定になりうる判定 {#search_judge_example}

無人の建物の中で、パソコンからデータを調査したい！
:   知力「コンピュータ」判定

ジャングルの中で、おいしい木の実を探したい！
:   知力「自然」判定

#### 会話判定 {#talk_judge}

NPCとの会話時に利用する。
言葉巧みに敵を騙したり、価値のある(少なくとも君たち研究員がそう思っている)物をいい値段で売ったりするためには、相応の話術が求められる。
そういう時に利用するのが「魅力」判定だ。
芸能技術で相手のココロをつかむことはもちろん、時にはホラや威圧で乗り切ることも大切になる。
こちらも、シナリオの展開次第では知力以外の判定を利用しても問題はない。

##### 会話判定になりうる判定 {#talk_judge_example}

お腹がすいたフレンズだ！何か作ってあげようかな……
:   魅力「料理」判定

普通の説得じゃどうしようもない……くそっ、こうなったらでまかせだ！
:   魅力「ペテン」判定

## 特殊な判定 {#special_example}

### 受動判定 {#passive_judge}

ダイスなしで算出される達成値。
主に相手の攻撃の回避や、目星を付けて何かを探すときなどに使われます。

回避判定
:    `(受動回避力) = 10 + (判断力修正値) + (回避値)`
    回避行動を先に取っていた場合は普通の判定と同様にダイスを振れます。

目星判定
:    `(受動目星) = 10 + (知力修正値) + (捜査値)`

### ダメージ判定 {#damage_judge}

ダメージ判定は、相手に攻撃がHITした際に、相手にどれだけの損害を与えたかを判定する。
達成度はそのまま相手に与えたダメージになるが、利用するダイスは武器に寄って異なる。
