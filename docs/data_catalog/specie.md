# バビルサ研究所ゲームパート概説(種族編) {#title}

2026/1/10

君たち研究員は、よほど特殊な事情がない限りは、アニマル・ガールであるはずだ。
そして、アニマル・ガールは、基となった動物に応じて利用できる能力が変わる。例えば、オオカミなどのイヌ科であればジャンプ力や嗅覚がより発達するし、タカなどの猛禽類であれば圧倒的な視力を身に着けている。
何物をも砕く剛腕を持つものもいるし、肉体そのものがどんな槍をもはじく鎧となるものもいる。
このような基礎レベルでの能力の違いは、この研究所では「種族」という形でデータベース化される(少なくともその予定だ)。
ここには、種族データベースに記載されるデータの意味と、ゲームパート上での影響について記載する。

## 分類共通データ {#common-data}

分類共通データは、種族の「分類」全体に共通する傾向であり、パラメータとその成長の傾向を決めるものが多い。

種族の分類を当てはめると自動で決定され、カスタマイズの余地はほとんど存在しない。
同種族同士の個性はあるかも知れないが、どうあがいても根本的な体力、知力、精神力及びそれらの成長はその種のあり方に縛られることになる。
小さなラーテルがライオンと殴り合うことは出来ても、ライオンそのものになることは出来ないのだ。

### パラメータ初期値 {#parameter-default}

種族レベル1の時点でのパラメータに付与されるボーナス値であり、パラメータごとに個別に設定される。
なお、レベル1時点のパラメータは、すべて`3d6+(パラメータ初期値)`で決定される。

### 必須習得技能 {#essential-skill}

種族レベル1の時点で習得している、技能パラメータの名称と成長量である。
これはアニマル・ガールが本能的に会得している基本的な技能を数値化したものでもある。

### 成長ダイス {#grow-dice}

種族レベルが上がった際の、HP最大値、技能ポイント、および各種基礎パラメータの成長量を決めるダイスを表す。
HP最大値と技能ポイントの成長ダイスはダイス一つ、ボーナスなしで設定されている。
基礎パラメータの成長ダイスはボーナス値も含めて設定されており、パラメータごとに別個に割り当てられている。
また、基礎パラメータの場合は「**全てのパラメータの期待値合計が6**」となるように設定されているため、個別で見るとむしろ弱化しかねないダイスが割り当てられている事もある。

## 種族差データ {#individual-data}

種族差データは、種族固有の特性やアニマル・ガールとしての個性を表す情報であり、環境への適性を決めるものが多い。
ただし、ここで言う「環境」とは、外的環境と内的環境を合わせた包括的な意味を持つ。

外的環境とは、気候や地形などといった、本人の意志でどうにもならない周囲の環境を表す。
例えば、一口にイヌと言っても寒帯のツンドラオオカミと熱帯のドールでは生まれ育った気候が大きく異なる。
本来の棲む世界が違えば、当然土の感触やまわりに立っている木の形、性質なども全く異なるはずだ。
そのような、基となる動物が生まれ育った環境こそが、ここで言う外的環境なのである。

一方、内的環境は、性格や特技などといった、本人自身の性質を表す。
元ヒトの研究員は実感するところだと思うが、同じ種族同士でも性格が全く異なることは珍しくない。
そして、性格の違いは、時として、外的環境によるストレスへの耐性に大きな影響を与える。
こと弱点においては、物理的なショックが苦手なケース、言葉責めが苦手なケース、成績に一喜一憂しやすいケースなどが想像しやすいところだろう。
このような、本人の個体としての個性や性質による環境への適性を、ここでは内的環境と呼ぶことにする。

これらのような個性はアニマル・ガールの数だけあるので、用意された各種データの中からある程度自由に決定することができる。
尤も、「熱帯気候に強いホッキョクグマ」などといった、明らかに基となる動物の生態と矛盾する個性は、なんらかの形で理由付けをする必要が出てくると思われるが。

### 気候適正 {#weather-appropriation}

アニマル・ガールが保有する、過酷な気候に起因するステータスへの影響を表す。
この適性は、外的環境に値する。

気候の適性は、温度と湿度の両軸で考える。
この軸はそれぞれ3段階に分けられており、2次元表で表すとこんな感じになる。

|湿度\\温度|熱帯|温帯|寒帯|
|:--:|:--:|:--:|:--:|
|**湿潤**|□|□|□|
|**普通**|□|□|□|
|**乾燥**|□|□|□|

アニマル・ガールの気候適性は、上の表の空白部分に一つの印をつける形で表現する。
例えば、暑くジメジメした熱帯地方に対する適性がある場合は、上の表の左上のマスにのみ印をつける。

|湿度\\温度|熱帯|温帯|寒帯|
|:--:|:--:|:--:|:--:|
|**湿潤**|■|□|□|
|**普通**|□|□|□|
|**乾燥**|□|□|□|

この場合、熱帯地方で仕事をすると大幅なバフを享受できるが、寒帯や夜の砂漠帯などの寒く湿度の低い環境ではえげつないデバフを受けてしまうことになる。

> 屋内や海上の気候は、基本的には「一番近い陸地と同等の気候」とすると設定しやすい。
> もちろん、シナリオ上で理由をつけて、独自に設定してしまっても構わない。

具体的な気候適正のルールは、以下に記す。

1. 気候適性は、1種類の組み合わせしか選べない
2. 真ん中の気候(温度は「温帯」、湿度は「普通」)を選択した場合は、その軸のすべての気候において一切の強化、弱化を受けない
    1. 例えば、湿度の適性が「普通」の場合は、湿度軸が「湿潤」でも「乾燥」でもとくに何も起こらず、温度の適正のみ判定される
3. 両端の気候のうちいずれかを選択した場合は……
    1. 選択した気候と一致する気候内にいる間ば、以下の強化が入る
        1. ダメージ判定以外のあらゆる判定の達成度に、1d6のボーナスがつく
        2. 最大HPが一時的に+1d10される
            1. ただし、最大HPが元に戻る際、HPの余剰分はカットされる
        3. 体内サンドスター濃度が1分ごとに1d4%回復(最大150%まで)
        4. 好きなタイミングで、2d6の無属性ダメージボーナスをつけられる(戦闘時限定、1回の戦闘につき1回まで)
    2. 選択した気候と真逆の気候内にいる間は、以下の弱化が入る
        1. 移動速度が-1m/10s(最低1m/10s、余剰分は他の要素による強化を打ち消す)
        2. HPが30分ごとに1d8減少
        3. ダメージを受ける際に、1d4の無属性追加ダメージを受ける
4. 3.の強化、弱化はそれぞれの軸ごとに判定、重複して発動する。例えば、「熱帯、湿潤」環境に適性がある状態で、
    1. 「熱帯、乾燥」環境：強化+弱化
    2. 「熱帯、湿潤」環境：強化重ねがけ(強化3.は、1回の戦闘につき2回発動できる)
    3. 「寒帯、普通」環境：弱化

### 地形適正 {#terrian-appropriation}

アニマル・ガールが保有する、地形に起因する活動への影響を表す。
この適性は、外的環境に値する。

地形適正のルールは以下に記す。

1. 研究員は、得意な地形を最大2つまで選択できる
2. 得意な地形を1つ以上選ぶ場合は、同じ数だけ苦手な地形を選択しなければならない
3. 得意な地形の上にいる間は、移動速度が+1m/10sされる
4. 苦手な地形の上にいる間は、移動速度が-1m/10sされる

以下、地形適正のラインナップ

- 土
- 草地
- 雪
- 泥
- 沼
- 砂
- 砂利
- 岩
- 濡れた岩
- 氷
- 木材
- 石材
- 金属板
- 水辺
- 水中
- 炎
- 縄
- セルリウム

### 移動速度 {#speed}

アニマル・ガールの移動速度である。
この適性は、内的環境に値する。

移動速度は種族ごとに平均値が設定されており、実際の研究員データには平均値に±1m/10sの誤差を付与することができる。
この誤差の値は1d3-2などで決定してもよいし、思い描くキャラクター性やゲーム上の運用を考えて、意図的に設定しても問題はない。

### 体質特性 {#physical-individuality}

アニマル・ガールの特異体質を表し、ゲーム上ではいわゆるパッシブ・スキルとして扱われる。
この適性は、内的環境に値する。

各種族には、体質特性が3種類(暫定)設定されており、その中から1つだけ選ぶことができる。
選ばないこともでき、その場合は後述するアニマル・スキルを選ばなかった数だけ余分に選択できる。
ただし、選ばなかった場合も含め、体質特性は付け替えることができないので、注意！

### 性格特性 {#mental-individuality}

アニマル・ガール自身の性格の傾向を表し、主に状態異常への耐性に影響する。
この適性は、内的環境に値する。

性格特性は、種族とは独立して6種類存在する。
研究員は、この中から自分に当てはまる性格特性を**必ず**指定数選択し、その弱点を適用しなければならない。

|性格名|イメージ|耐性|弱点|
|:--:|--|--|--|
|ファニー|おもしろい、話して楽しい|恐怖、風邪、くらくら|混乱、ガンギマリ、出血|
|フレンドリー|朗らか、親しみやすい、天真爛漫|魅了、封印、ちぐはぐ|怒り、くらくら、石化|
|リラックス|思慮深い、やさしい、落ち着きある|怒り、出血、ガンギマリ|火傷、封印、風邪|
|ラブリー|愛嬌、純真、幼い|石化、怒り、封印|魅了、恐怖、ちぐはぐ|
|アクティブ|明朗快活、勇敢、頼りになる|火傷、ちぐはぐ、混乱|氷結、怒り、封印|
|マイペース|のんびり屋、不思議ちゃん|恐怖、混乱、怒り|睡眠、ちぐはぐ、くらくら|

以下が細かいルールとなる。

1. 基本
    1. 耐性のある状態異常は、付与される際に相手の命中判定に1d6のペナルティを付与する。
    2. 耐性がダブると、付与される際の相手の判定は自動失敗となる。
    3. 弱点状態異常は、付与される際に相手の命中判定に1d6のボーナスを付与する。
    4. 弱点がダブった場合は、付与される際の相手の判定が自動成功してしまう。
2. 純アニマル・ガールの場合
    1. 1つの性格特性を選択する。
    2. 選択した性格特性と対応する状態異常が、そのまま耐性、弱点になる。
3. 元ヒトアニマル・ガールの場合
    1. 素体のヒトが持つ性格特性と、変身語のアニマル・ガールが持つ性格特性を、それぞれ一つずつ選択する。
    2. 初期状態では、ヒト側の性格特性の耐性、弱点が適用される。
    3. 人間度が60%, 40%, 20%を切るごとに、一つのヒト側の耐性がアニマル・ガール側の弱点に変換される。
    4. 人間度が閾値を上回ったら、逆に一つのアニマルガール側の弱点がヒト側の耐性に変換される。
    5. また、任意のタイミングで人間度を2d5%減少させることで、一つのヒト側の弱点をアニマル・ガール側の耐性に変換できる。
    6. いずれの場合も、変換される耐性、弱点は、無効なものからランダムに抽選される。
    7. そして人間度が0%になると……、完全にアニマル・ガールの耐性、弱点が解放される。

### 任意習得技能 {#optional-skill}

種族レベル1の時点で習得している、技能パラメータの名称と成長量を表す。
ただし、必須習得技能と異なり、種族ごとに6種類ある習得技能から3種類の習得したい技能を選択する必要がある。
この適性は、内的環境に値する。

### アニマル・ウェポン {#animal-weapon}

早い話、このゲームにおけるアニマル・ガールにとってのメインウェポンである。
アニマル・ガールには、生まれつき武器を身につけている個体も数多く存在する。
そうでなくとも、ほとんどのケースにおいて、そもそもの膂力自体アニマル・ガールはヒトより遙かに強い。
そのようなアニマル・ガールが、我々のイメージする「素手」の代わりに使うモノが、このアニマル・ウェポンなのである。

アニマル・ウェポンは種族ごとに3種類存在し、そこから1つを選択して戦闘に出かける。
セッション中でなければ、任意に付け替えも可能だ。いろんなアニマル・ウェポンを試してみよう。

ちなみに、「役職」によってはヒトが作り出した武器を利用した方が強いケースもある。
そのような武器は、アニマル・ウェポンとの区別のために「サブウェポン」と呼ばれる。

### アニマル・ムーブ {#animal-move}

アニマル・ガールの本能に刻まれたスキルを表す、種族由来のスペシャル・ムーブである。
基本的には種族内に10種類存在しており、発動条件さえ整っていればすべてのアニマル・ムーブをすぐに利用できる。
ただし、発動条件には体内サンドスター濃度だけでなく、人間度が一定値以下であることも含まれることが多い。
人間度の条件が満たされていなければ、能力が弱化したり、そもそも使えなかったりする。
利用する際はしっかり目を配ろう。

### 特殊アニマル・ムーブ {#special-animal-move}

「Cellien Killer」の上級役職「Natural」のみが習得できる、アニマル・ムーブを意識的に研鑽することで会得した、より強力なスペシャル・ムーブである。
