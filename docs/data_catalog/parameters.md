# バビルサ研究所ゲームパート概説(ステータス・パラメータ編) {#title}

2024/11/25

---

## データ {#data}

### ステータス {#status}

HP
:   ヒット・ポイント。
    あらゆる生物の体力を表す数値。
    一般的なRPGにおけるヒット・ポイントと同様、ダメージによって減少していく。
    アニマルガールの所有するこの値が0になるとギブアップ状態となり行動不能になる。
    この値はロストに関与しないが、0になっても戦闘から完全に離脱できる訳でもない点に注意。
    なお、このステータスのみ成長に応じて最大値が変動する。
    計算は以下の通り。

    `HP=(ヒット・ダイスの値合計)+(現在Lv)*(耐久力修正値)`

SS
:   体内サンドスター濃度。
    あらゆる生物が内包するサンドスターの含有量。
    一般的なアニマルガールの所有するサンドスターを100％とし、相対量で表す。
    アニマルガールの所有するこの値が0になると、研究者NPC問わずそのキャラはロストする。
    また、基本的にスキルのコストにもなる。

人間度
:   所有者の人間としての記憶や自我。
    基本的に人間は100％、純アニマルガールは0％。
    この値によって利用可能なスキルや武器が変わる。
    基本的に探索の結果によって変動するほか、アイテムによって意図的に値を変更することも可能。

移動速度
:   10秒間に移動出来る距離。
    特にハウスルールが適用されていない場合は、簡易戦闘やクラシック戦闘において、1ターンに動けるマスの数でもある。
    単位は「m/10s」。

### 非ステータスデータ {#not_status}

Exp
:   経験値。
    そのフレンズの研究員としての経験量を表す。
    一定の値になるごとにそのキャラのLvが変動する。

Lv
:   レベルとも呼ばれる。
    その研究員の現在の「役職」におけるランク。
    これが変動するごとに、研究員は以下のボーナスを得る。

    - 種族に合わせた基礎ステータスの強化
    - 上記に付随する最大HPの強化
    - 特殊ステータスを強化するためのボーナス値
    - 新しいクラススキル

所持金
:   この世界で使えるお金。
    基本的にクエストの報酬としてこれが支払われる他、アイテムのやりとりでも増減が発生することがある。
    また、研究員間で個別にやりとりも可能。

    この世界では産業技術の大半が失われている影響で物資の価値が全体的に高い。
    そのせいもあってお金の価値はややインフレ気味だったりする。

## 基礎パラメータ・技能 {#parameter}

基礎パラメータは、攻撃、防御、探索、会話で表される、4種類の使用用途ごとに明確に分離されている。
判定時にはこれらのパラメータより算出された「修正値」と、D20の出目の値を足し合わせたものを達成値としなければならない。

`修正値=(パラメータの値/2)-5`

各パラメータが具体的に何を表すかは以下の通りである。

### 攻撃パラメータ {#attack_parameter}

戦闘に利用するパラメータのうち、各種攻撃の命中判定に利用する判定。
一部のデバフ付与攻撃などもこちらに属している。
全3種

筋力
:   所持するアニマルガールの、運動神経に寄らない純粋な膂力の強さを表す。
    この値が高いフレンズは、物質を破壊したり、空高くまで重いものを投げ飛ばしたりすることができる。
    ただし、このパラメータが高いからと言って、そのフレンズが運動のセンスに長けているとは限らない。
    至近距離での格闘や、ものをブン投げての遠距離攻撃などに利用する。

このパラメータが重要になる「役職」

- セルリアンキラー
- ウェポンマスター

敏捷力
:   所持するアニマルガールの運動神経や動体視力を表す。
    この値が高いフレンズは、先手を打って動きだす、高速で動く相手を目で追って捕縛するなどの動作に長ける。
    射撃武器での命中判定やイニシアチブ判定などに使う。

このパラメータが重要になる「役職」

- ウェポンマスター
- アルケミスト

魔力
:   所持するアニマルガールの、けものプラズムの取り扱いのうまさを表す。
    この値が高いフレンズは、自身や大気中のけものプラズムを応用して、無からエネルギーや物質を生み出すなどの魔術めいた事象を発動させたり、動物だったころの能力を容易く行使したりすることができる。
    一部のダメージ系スキル、ほとんどの状態異常付与攻撃の命中判定に使う。

このパラメータが重要になる「役職」

- セルリアンキラー
- ジオロジスト

### 防御パラメータ {#deffence_parameter}

戦闘に利用するパラメータのうち、状態異常を含めた相手の攻撃からの回避判定に利用するもの。
全3種

判断力
:   そのフレンズの瞬間的な判断力を表す。
    とっさの回避にはもちろん、重い盾や鎧を身に着けて防御力に特化させていてもこの値は利用する。
    また、この値が高いフレンズは、戦況を俯瞰し適切な行動をする・させる能力に長けている。
    通常攻撃とメンタル系状態異常の回避・抵抗に利用する。

このパラメータが重要になる「役職」

- **ほぼすべて**

耐久力
:   そのフレンズの肉体的な頑丈さを表す。
    この値が高いフレンズは、他のフレンズよりも体力や生命力が高くなり、病気にも若干なりづらくなる。
    フィジカル系状態異常、シックネス系状態異常への抵抗に利用するほか、前述通りHPはこの値をベースに算出される。

このパラメータが重要になる「役職」

- セルリアンキラー
- ウェポンマスター

運

:   そのフレンズの悪運の強さ。
    この値が高いフレンズは、どういうわけか致命的なな状況を回避、突破できる可能性が上がる。
    カース系状態異常への抵抗やトラップの回避などで利用するほか、ＧＭ判断で失敗するとロストのリスクがある判定への保険の判定として追加するのにも適している。

このパラメータが重要になる「役職」

- アルケミスト
- ローグ

### 探索パラメータ {#searc_parameter}

探索時、特にアイテムやギミックの発見・動作時に使用するパラメータ。
全1種。

知力
:   そのフレンズの知識量を表す。
    このパラメータは「技能」としてフレンズごとの得意分野によって細分化されており、探索するアイテムやギミックによってどの技能を利用するか決定することができる。

    例：

    - 倒れた木の形跡から災害の内容を導き出したい
    →「自然」技能を利用
    - 壊れた巨大なマシンを修理したい
    →「機械工学」
    - セルリアンの特性を知りたい
    →「サンドスター学」

    ……など

    一覧：

    - 自然
    - オカルト
    - 歴史
    - 機械工学
    - 地政学
    - コンピュータ
    - 捜査
    - 言語
    - サンドスター学
    - 医学
    - 禁忌

このパラメータが重要になる「役職」

- ジオロジスト
- アルケミスト

### 会話パラメータ {#talk_parameter}

NPCとの会話時に、相手から効率よく情報を聞き出し、イベントを発生させるためのフラグを建てるのに利用できる技能。
全1種。

魅力
:   コミュニケーション能力や芸能技術など、そのフレンズの外見に寄らない魅力を表す。
    こちらも「技能」として細分化されており、披露したい能力によって利用する技能を決める。

    例：
    - 悲しみに暮れるヒトを即興の水族館ショーで元気づけたい
    →「曲芸」
    - 自分の体験談を詩に表して遠くのヒトに伝えたい
    →「作詞」
    - いきり立つ暴徒を落ち着かせたい
    →「威圧」または「説得」
    ……など

    一覧：

    - 威圧
    - 歌唱
    - 曲芸
    - 作詩
    - 楽器
    - 説得
    - 料理
    - ペテン

このパラメータが重要になる「役職」

- ローグ
