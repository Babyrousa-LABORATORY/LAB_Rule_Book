# アイテムの種類 {#title}

## 装備品について {#about_equipable}

装備できるアイテムは、「サブウェポン」「盾」「服」「鎧」「アクセサリー」「消耗品」「だいじなもの」の7種類。
サブウェポンと盾は択一、服及び鎧はいずれか1種類まで、アクセサリーは服、鎧と合わせて4種類まで装備することが可能。
また、いずれのアイテムも、装備枠とは別口で最大10枠分キャラ内に保持することができる。
それら以外のアイテムは「倉庫」に送られることとなり、セッション中は利用できない。
アイテムには最大スタック数が設定されており、その範疇であれば複数個のアイテムを1枠分にまとめることもできる。

「サブウェポン」は専用のDBに、「盾」「鎧」「アクセサリー」「消耗品」はアイテムDBにまとめられている。
要チェックだ。

### サブウェポン {#sub_weapon}

主武器となる「フレンズウェポン」とは別途装備可能な武器。
基本となる武器は大まかにわけて格闘武器、投擲武器、射撃武器の3種類に分類することができる。
これらはアニマルガールが普通に振り回すだけでもある程度の効果を発揮できるが、ウェポンマスターなどの武器へ習熟した者に使わせると最大クラスの効果を発揮する。
また、ウィップ、ブーメランなど、一部のサブウェポンは特殊効果を発動することがある。

#### 格闘武器 {#melee_weapon}

格闘判定で利用できる武器。
基本的に近距離格闘では最もリーズナブルな選択肢となり、取り扱いもたいがいのフレンズウェポンとほぼ遜色なく利用することができる。

ハマー
:   メイス、棍、金槌など。
    最も原始的な武器故に、フレンズウェポンとの相性もよいことが多い。

ソード
:   ショートソードから日本刀まで幅広い武器を意味する。
    コスパが良さと洗練されたフォルムから、ウェポンマスターの中でも人気の高い武器。

スピア
:   槍、薙刀、ハルバードなど。
    刺突属性を持つ格闘武器。
    基本的に長射程が正義である白兵戦では、最も需要が高い武器の一つ。
    とはいえ、「突く」という動作があまり一般的ではないためか、これを使いこなせるフレンズは上級ウェポンマスターとも呼ばれる。

ウィップ
:   鞭、モルゲンシュテルンなど。
    材質は木材や鎖など多岐にわたり、基本的に殺傷力は低いとされている。
    当然例外は存在するのだが……動物の毛皮を利用していたり、大量のヒトの血を吸っていたりと、なにかと曰く付きのものが多い。
    また、適正射程外から近すぎても遠すぎても、火力ががた落ちする特性がある。
    取り扱いはかなり難しい部類に当たるだろう。

#### 投擲武器 {#throw_weapon}

投擲判定で利用できる武器。
ぶん投げて使用する遠距離攻撃の一つ。
一つ一つの値段が安価であるが、このタイプの武器のみアイテムとして消費して利用しなければならない。
また、投げるという挙動が動物のころにはなかった挙動であるためか、セルリアンキラーには使うのも使われるのも苦手とする者も多い。

ストーン
:   最も原始的な投擲武器が一つ。
    練度がない者にも扱いやすいうえ、そこら辺の石ころを拾い上げるだけで有効打を与えられる。
    武器に精通していない者にとっては、下手に店売りの品を購入するよりもそこら辺の石ころの方がコスパがよいことも少なくない。

ハンドアクス
:   トマホークやフランキスカなど、ストーンよりは武器として洗練された投擲武器。
    投げやすく、振り回すことで緊急回避にも利用できるため、ウェポンマスターはこれ＋格闘武器1種類で立ち回ることが一般的である。

スローイングナイフ
:   文字通り、投げナイフ。
    格闘武器としては小さく、投擲武器としてもハンドアクスと比べて標的に刺すには若干のコツが必要。
    しかし、慣れると小さいからこそ手数をそろえられ、軽量故に遠くまで飛ばせるオールラウンダーとして活躍する。

ブーメラン
:   投げると戻ってくる飛棍。
    行きと帰りで最大2回まで命中判定を振れる上、いずれにも失敗した場合は武器を消費しなくてよいという特性を持つ。
    ただし、何らかのサンドスターエネルギーがかかっていない限りは、行きと帰りでダブルヒットなどはしないので注意。

#### 射撃武器 {#shoot_weapon}

射撃判定で利用できる武器。
身もふたもなく言えば弓矢や銃火器。
これらには狩猟用武器のイメージが未だに根強く残っており、射撃武器というだけで嫌うアニマルガールも多い。
しかし、ある程度の射程と高火力を両立していることも珍しくなく、名誉を含めた初期投資が極端に高くつくことを除けば、理想的な武器となる……かもしれない。
刺突属性を持つことが多い。

ボウ
:   弓矢。
    取扱いに相応の筋力が必要だが、アニマルガールであれば弓を引き絞るぐらい問題なくできるだろう。
    また、当然ながら投擲武器と比較して遙かに長い射程を持つため、射手らしい戦術を展開できる数少ない武器である。

クロスボウ
:   ボルトを飛ばすメカニカルな弓矢。
    高威力な傾向が強いが、若干かさばる上にリローディングに妙に時間がかかるのが難点。

ハンドガン
:   リボルバー式の拳銃。
    一度に複数の弾丸を込められること以外は、ゲーム的にはクロスボウと大差はない。
    お好みでどうぞ。
    ……しかし、ハンドガンの有効射程は想定以上に短いらしい。

ショットガン
:   ある意味、禁断の武器。
    射程を犠牲にした代わりに、火力に全てのステータスを振った最終兵器。
    ……というのが通説だが、実のところ射程も大して犠牲になっておらず、20mぐらいまでなら余裕で届いたりするし、なんなら100mでも充分危ない。
    当然、理論上最強の武器だが、この世界では猟銃としての悪名が浸透してしまっており、これを装備しているだけで善良な市民から色眼鏡で見られてしまうことも珍しくないのだとか。

#### その他 {#other_weapon}

上記の武器に当てはまらない武器。
基本的に何かしらの特殊効果が備わっている者が多く、その分使いこなすのも一筋縄ではいかない曲者も多い。

### 盾 {#shield}

サブウェポンと択一で装備できる装備品。
回避判定の代わりに使用することで、相手の攻撃を判定なしで全カットできるという強力な特性を持つ。
ただし、使用に際しては以下のルールを守らなければならない。

>- 回避判定の保険として盾を利用することはできない。
>- 盾には耐久値が存在し、カットした分のダメージ量だけ盾の耐久値を消費する。
>- 盾には耐性と弱点が存在しており、これらに当てはまる属性の攻撃をカットした場合は消耗する耐久値が半分または2倍になる。
>- 盾の耐久値が0になったらその盾は破壊され、同じシナリオにおいて利用ができなくなる。
>- 受けるダメージが盾の残り耐久値を上回った場合は、本来のダメージから盾の残り耐久値を引いた分だけダメージを受ける。
>- 盾の耐久値をシナリオ中に回復することはできない。

### 服、鎧、アクセサリー {#armor}

今作における「服」「鎧」「アクセサリー」は、ダメージをカットするものではなく、回避力や特技を補強する装備品として存在している。
これら装備には、回避を初めとした各種特殊パラメータに対するボーナスが割り当てられており、その値の分だけ回避技能および受動回避力に+の補正がつく。

このゲームにおいては、「お出かけ服」などのごく一部の例外を除き、軽装と重装の大きな違いは存在しない。
また、一部の敵によって鎧が破壊された場合は、盾とことなり再利用に原価の半分の値段を消費し修理を依頼しなければならない。

### 消耗品 {#consumable}

「消耗品」は、戦闘中または探索中に使用することで効果を発揮するアイテム。

基本的には1回利用すると消失するが、中には一定回数まで複数回利用可能なアイテムや、~~消耗品と言いつつ~~何度使ってもなくならないものもある。
また、一部の武器や盾、アクセサリーなども消耗品として利用すると効果があったりする。

...まあ要するに、君が単に「アイテム」と聞いて思い浮かべるものと思えば良いだろう。
ただし、消耗品の中にはそもそも明確な使い方が定義されていないものも存在する。
それらのアイテムは、君がセッションをこなす中で使い道を見出さなければならない。
君が「うまい」使い方をすれば、絶望的な状況を打破することもきっとできるはずだ。

### だいじなもの {#importance}

「だいじなもの」は、GMが設定した物語のキーになるアイテム。
どのようなアイテムかは完全にGMが制定することができるが、別シナリオへの持ち出しは基本的に想定していない。
例外があるとすれば、同じGMが続編として次のシナリオを提出する想定がある場合が考えられるが、その場合はGMがセッション終了時に、引き継ぐだいじなものを一旦預かることが望ましい。
